# Codzenith_Weather-API
A simple and responsive Weather Application developed as part of my internship assignment in Codzenith. The application fetches real-time weather information for any city using a weather API.
<br>
## Features
- Search weather by city name
- Displays temperature, humidity, wind speed, and weather conditions
- Responsive user interface
- Real-time weather updates

## Tech Stack
- HTML
- CSS
- JavaScript
- React JS
- Weather API

## Project Purpose

This project was developed as an internship assignment to strengthen my frontend development skills and gain hands-on experience with API integration.
